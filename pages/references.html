No botão abaixo, você confere a situação de compatibilidade dos navegadores das tags listadas.


<a> Define um hyperlink
<bdo> A tag <bdo> é usada para substituir a direção do texto atual.
<body> A tag <body>representa o conteúdo de um documento HTML.
<br> A tag <br> insere uma única quebra de linha.
<datalist> A tag <datalist> especifica uma lista de opções predefinidas para um elemento <input>.
<del> A tag <del> é usada para indicar texto deletado ou removido em uma página HTML.
<div> A tag <div> cria uma divisão ou seção em um documento HTML que é usada como um contêiner para elementos HTML.
<dl> A tag <dl> engloba uma lista de pares de termos e descrições.
<dt> O elemento HTML <dt> (ou Elemento HTML de Definição de Termo) identifica um termo na lista de definição.
<fieldset> A tag <fieldset> é usada para agrupar elementos relacionados em um formulário.
<footer> A tag <footer> define um rodapé para um documento ou seção.
<head> A tag <head>providencia informações gerais (metadados) sobre o documento, incluindo seu título e links para scripts e folhas de estilos.
<header> A tag <header> é usada para definir o cabeçalho de uma seção ou de um artigo em um documento HTML.
<i> A tag <i> é frequentemente usada para indicar um termo técnico, uma frase de outro idioma, um pensamento, um nome de navio, etc..
<link> O Elemento HTML <link> especifica as relações entre o documento atual e um recurso externo.
<main> A tag <main> especifica o conteúdo principal de um documento.
<map> A tag <map> é usada para definir um mapa de imagem.

<ol> A tag <ol> define uma lista ordenada.
<option> A tag <option> é usada em um elemento <select> para definir as opções de escolha que um usuário pode selecionar.
<p> A tag <p> é usada para definir um parágrafo.
<param> A tag <param> é usada para definir parâmetros para um elemento <object>.
<picture> A tag HTML <picture> permite exibir diferentes imagens com base no dispositivo ou tamanho da tela.
<select> O <select> elemento é usado para criar uma lista suspensa.
<strong> A tag <strong> é usada para definir texto com forte importância. O conteúdo interno geralmente é exibido em negrito.
<svg> A <svg>tag define um contêiner para gráficos SVG.
<tbody> A tag <tbody> é usada para agrupar o conteúdo do corpo em uma tabela HTML.
<tfoot> A <tfoot>tag é usada para agrupar o conteúdo do rodapé em uma tabela HTML.
<thead> A tag <thead> é usada para agrupar o conteúdo do cabeçalho em uma tabela HTML.

<ul> A tag <ul> define uma lista não ordenada (marcada).
