Tag <tbody>

Definições e uso

A tag <tbody> é usada para agrupar o conteúdo do corpo em uma tabela HTML. O elemento <tbody> é usado em conjunto com os elementos <thead> e <tfoot> para especificar cada parte de uma tabela (corpo, cabeçalho, rodapé).

Os navegadores podem usar esses elementos para permitir a rolagem do corpo da tabela independentemente do cabeçalho e do rodapé. Além disso, ao imprimir uma tabela grande que abrange várias páginas, esses elementos podem permitir que o cabeçalho e o rodapé da tabela sejam impressos na parte superior e inferior de cada página.

Nota: O elemento <tbody> deve ter uma ou mais tags <tr> dentro dele.

A tag <tbody> deve ser usada no seguinte contexto: Como filho de um elemento <table>, após qualquer elemento <caption>, <colgroup> e <thead>.

Dica: Os elementos <thead>, <tbody> e <tfoot> não afetarão o layout da tabela por padrão. No entanto, você pode usar CSS para estilizar esses elementos (veja o exemplo abaixo)!

Exemplo

                        
<!DOCTYPE html>
<html>
    <head>
        <title> Tag tbody</title>
    </head>
    <body>
        <h1>Exemplo de uso da tag tbody</h1>
        <table>
            <thead>
                <tr>
                <th>Mês</th>
                <th>Economia</th>
                </tr>
              </thead>
            <tbody>
                <tr>
                <td>Janero</td>
                <td>$100</td>
                </tr>
              <tr>
                <td>Fevereiro</td>
                <td>$80</td>
                </tr>
              </tbody>
            <tfoot>
                <tr>
                <td>Soma</td>
                <td>$180</td>
                </tr>
              </tfoot>
            </table>
    </body>
</html>

Atributos Padrões

Atributo Valor Descrição
onafterprint script Script a ser executado após a impressão do documento
onbeforeprint script Script a ser executado antes da impressão do documento
onbeforeunload script Script a ser executado quando o documento estiver prestes a ser descarregado
onerror script Script a ser executado quando ocorre um erro
onhashchange script Script a ser executado quando houver alterações na parte âncora de uma URL
onload script Dispara depois que a página termina de carregar
onmessage script Script a ser executado quando a mensagem é acionada
onoffline script Script a ser executado quando o navegador começar a funcionar offline
ononline script Script a ser executado quando o navegador começar a funcionar online
onpagehide script Script a ser executado quando um usuário sai de uma página
onpageshow script Script a ser executado quando um usuário navega para uma página
onpopstate script Script a ser executado quando o histórico da janela mudar
onresize script Dispara quando a janela do navegador é redimensionada
onstorage script Script a ser executado quando uma área de armazenamento da Web é atualizada
onunload script Dispara quando uma página é descarregada (ou quando a janela do navegador é fechada)
