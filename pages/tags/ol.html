Tag <ol>

Definições e uso

A tag <ol> define uma lista ordenada. Uma lista ordenada pode ser numérica ou alfabética.

Exemplo

                        
<!DOCTYPE html>
<html>
    <head>
        <title> Tag ol</title>
    </head>
    <body>
    <p><i> 2023 - © Trabalho de Conclusão de Etapa</i></p>

    </body>
</html>

Atributos Padrões

Atributo Valor Descrição
reversed reversed Especifica que a ordem da lista deve ser invertida (9,8,7...)
start number Especifica o valor inicial de uma lista ordenada
type 1
A
a
I
i		 Especifica o tipo de marcador a usar na lista
