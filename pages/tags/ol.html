Tag <ol>

Definições e uso

A tag <ol> define uma lista ordenada.
Uma lista ordenada pode ser numérica ou alfabética.
Use a tag <ol> junto com a tag <li> para criar listas ordenadas.

Exemplo

                        
<!DOCTYPE html>
<html>
    <head>
        <title>Tag ol</title>
    </head>
    <body>
    <h1>O elemento ol</h1>

    <ol>
    <li>Café</li>
    <li>Chá</li>
    <li>Leite</li>
    </ol>
    
    <ol start="50">
    <li>Café</li>
    <li>Chá</li>
    <li>Leite</li>
    </ol>
    
    </body>
</html>

Atributos Padrões

Atributo Valor Descrição
reversed reversed Especifica que a ordem da lista deve ser invertida (9,8,7...)
start number Especifica o valor inicial de uma lista ordenada
type 1
A
a
I
i		 Especifica o tipo de marcador a usar na lista
