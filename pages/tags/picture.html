Tag <picture>

Definições e uso

A tag HTML <picture> permite exibir diferentes imagens com base no dispositivo ou tamanho da tela. Ela é usada para fornecer imagens otimizadas para diferentes cenários, melhorando o desempenho e a qualidade visual.
Contém duas tags apenas, duas ou mais tags <source> e uma tag <image>.
A função do elemento <source> é indicar ao navegador quais imagens estão disponíveis e qual delas deve ser usada, dependendo das condições de mídia especificadas. Se nenhuma das tags <source> atender às condições, o navegador usará a imagem definida na tag <image> dentro da tag <picture> como fallback.

Exemplo

                    
<!DOCTYPE html>
<html>
<head>
  <title>Exemplo de Tag Picture</title>
</head>
<body>
  <h1>Exemplo de Tag Picture</h1>
                           
  <picture>
    <source srcset="imagem-grande.jpg" media="(min-width: 1200px)">
    <source srcset="imagem-media.jpg" media="(min-width: 800px)">
    <source srcset="imagem-pequena.jpg">
    <img src="imagem-padrao.jpg" alt="Imagem de exemplo">
  </picture>
                           
</body>
</html>
