Tag <option>

Definições e uso

A tag <option> é usada em um elemento <select> para definir as opções de escolha que um usuário pode selecionar. A tag <select> permite que o usuário selecione uma ou várias opções de uma lista pré-definida de valores. As opções são definidas dentro da tag <select> usando a tag <option>. Cada tag <option> representa uma opção na lista suspensa, que pode ser selecionada pelo usuário. O atributo "value" da tag <option> define o valor associado à opção selecionada, enquanto o texto dentro da tag <option> é exibido como o rótulo da opção na lista suspensa. <p>

Exemplo

                        
<!DOCTYPE html>
<html>
    <head>
        <title>Exemplo de tag <option></title>
    </head>
    <body>
        <h1>Escolha uma fruta:</h1>
        <select name="fruits">
            <option value="apple">Maçã</option>
            <option value="banana">Banana</option>
            <option value="orange">Laranja</option>
            <option value="grape">Uva</option>
        </select>
    </body>
</html>

Atributos Padrões

Atributo Valor Descrição
disabled disabled Especifica que uma opção deve ser desabilitada
label text Especifica uma etiqueta mais curta para uma opção
selected selected Especifica que uma opção deve ser pré-selecionada quando a página é carregada
value text Especifica o valor a ser enviado para um servidor
