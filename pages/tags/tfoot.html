Tag <tfoot>

Definições e uso

A <tfoot>tag é usada para agrupar o conteúdo do rodapé em uma tabela HTML.
O <tfoot>elemento é usado em conjunto com os elementos <thead> e <tbody> para especificar cada parte de uma tabela (rodapé, cabeçalho, corpo).

Exemplo

                    
<!DOCTYPE html>
<html>
<head>
  <title>Tabela com Rodapé</title>
</head>
<body>
  <h1>Tabela com Rodapé</h1>
                       
    <table>
        <thead>
            <tr>
                <th>Produto</th>
                <th>Quantidade</th>
                <th>Preço Unitário</th>
            </tr>
        </thead>
        <tbody>
            <tr>
                <td>Item 1</td>
                <td>5</td>
                <td>$10</td>
            </tr>
            <tr>
                <td>Item 2</td>
                <td>3</td>
                <td>$15</td>
            </tr>
        </tbody>
        <tfoot>
            <tr>
                <td colspan="2">Total</td>
                <td>$55</td>
            </tr>
        </tfoot>
        </table>
                       
        </body>
</html>
