Tag <div>

Definições e uso

A tag <div> cria uma divisão ou seção em um documento HTML que é usada como um contêiner para elementos HTML.
Para estilizar a tag <div>, pode-se usar os atributos class ou id. Além disso, a tag <div> é versátil, pois é possível inserir qualquer tipo de conteúdo dentro dela.

Exemplo

                        
<!DOCTYPE html>
<html>
    <head>
        <title>Exemplo de tag <div>
        <style>
            .container {
            background-color: #f2f2f2;
            padding: 10px;
        }
        </style>
    </head>
    <body>
        <h1>Meu site</h1>
        <div class="container">
            <p>Este é um exemplo de uso da tag <div>. Qualquer tipo de conteúdo pode ser colocado dentro dela.</p>
            <ul>
                <li>Item 1</li>
                <li>Item 2</li>
                <li>Item 3</li>
            </ul>
         </div>
    </body>
</html>
