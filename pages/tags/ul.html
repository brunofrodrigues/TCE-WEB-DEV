Tag <ul>

Definições e uso

A tag <ul> define uma lista não ordenada (marcada).
Use a tag <ul> junto com a tag <li> para criar listas não ordenadas.

Exemplo

                        
<!DOCTYPE html>
<html>
    <head>
        <title>Tag ul</title>
    </head>
    <body>
    <h1>O elemento ul</h1>

    <ul>
    <li >Café</li>
    <li >Chá</li >
    <li >Leite</li >
    </ul>

    </body>
</html>
