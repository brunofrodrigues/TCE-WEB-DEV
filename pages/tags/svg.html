Tag <svg>

Definições e uso

A <svg>tag define um contêiner para gráficos SVG.

Exemplo

                    
<!DOCTYPE html>
<html>
<head>
  <title>Exemplo de Tag SVG</title>
</head>
<body>
  <h1>Exemplo de Tag SVG</h1>
                       
  <svg width="200" height="200">
    <circle cx="100" cy="100" r="50" fill="red" />
  </svg>
                       
</body>
</html>
