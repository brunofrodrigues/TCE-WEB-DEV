Tag <dl>

Definições e uso

A tag <dl> engloba uma lista de pares de termos e descrições.

Exemplo

                        
<!DOCTYPE html>
<html>
    <head>
        <title> Tag dl</title>
    </head>
    <body>
        <h1>Example of using the dl Tag</h1>
            <p>These three elements are used to create a description list:</p>
            <dl>
            <dt>Coffee/dt>
            <dd>Black hot drink</dd>
            <dt>Milk
            <dd>White cold drink</dd>
        </dl>
    </body>
</html>
