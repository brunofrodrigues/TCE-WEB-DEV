Tag <main>

Definições e uso

A tag <main> especifica o conteúdo principal de um documento.

O conteúdo dentro do <main> elemento deve ser exclusivo do documento. Ele não deve conter nenhum conteúdo repetido em documentos, como barras laterais, links de navegação, informações de direitos autorais, logotipos de sites e formulários de pesquisa.

Nota: Não deve haver mais de um <main> elemento em um documento. O <main>elemento NÃO deve ser descendente de um elemento <article>, <aside>, <footer>, <header> ou <nav>.

Exemplo

                        
<!DOCTYPE html>
<html>
<body>
<h1> O elemento main </h1>
<main>
    <h1> Navegadores mais populares </h1>
        <p> Chrome, Firefox, e Edge são os navegadores mais usados atualmente </p>
    <article>
    <h2>Google Chrome</h2>
        <p> O Google Chrome é um navegador da web desenvolvido pelo Google, lançado em 2008. 
        O Chrome é o navegador da web mais popular do mundo hoje</p>
            <p>Isto é um parágrafo</p>
    </article>

    <article>

    <h2>Mozilla Firefox</h2>
    <p> Mozilla Firefox é um navegador web de código aberto desenvolvido pela Mozilla. 
    O Firefox é o segundo navegador mais popular desde janeiro de 2018.</p>
    </article>

    <article>
    <h2>Microsoft Edge</h2>
    <p> O Microsoft Edge é um navegador da Web desenvolvido pela Microsoft, lançado em 2015. 
    O Microsoft Edge substituiu o Internet Explorer.</p>

    </article>
    </main>
    </body>
    </html>
