Tag <header>

Definições e uso

A tag <header> é usada para definir o cabeçalho de uma seção ou de um artigo em um documento HTML. Geralmente, é usado para colocar títulos, sub-títulos, logotipos ou outros elementos que sejam relevantes para a seção ou para o artigo. O cabeçalho geralmente aparece no topo da página e ajuda a identificar o conteúdo da seção ou do artigo. Além disso, ele pode ser estilizado com CSS para torná-lo mais atraente ou para melhorar a navegação no site. É importante notar que a tag "header" não deve ser confundida com o elemento "head", que é usado para armazenar informações do documento, como metadados, scripts e folhas de estilo.

Exemplo

                        
                            <!DOCTYPE html>
                            <html>
                            <head>
                            <title>Exemplo de uso da tag header</title>
                            </head>
                            <body>
                            <header>
                            <h1>Título da Página</h1>
                            <nav>
                            <ul>
                            <li><a href="#">Página Inicial</a></li>
                            <li><a href="#">Produtos</a></li>
                            <li><a href="#">Contato</a></li>
                            </ul>
                            </nav>
                            </header>
                            <main>
                            <p>Conteúdo principal da página vai aqui.</p>
                            </main>
                            <footer>
                            <p>Rodapé da página vai aqui.</p>
                            </footer>
                            </body>
                            </html>

Definições e uso

A tag <thead> é usada para agrupar o conteúdo do cabeçalho em uma tabela HTML.

Exemplo

                        
<!DOCTYPE html>
<html>
<table>
    <thead>
        <tr>
            <th>ID</th>
            <th>Produto</th>
            <th>Preço</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>1</td>
            <td>Melancia</td>
            <td>R$30</td>
        </tr>
    </tbody>
    <tfoot>
    </tfoot>
  </table>
</html>
