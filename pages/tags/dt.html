Tag <dt>

Definições e uso

A tag <dt>O elemento HTML <dt> (ou Elemento HTML de Definição de Termo) identifica um termo na lista de definição. Este elemento pode ocorrer somente em um elemento filho de <dl>. Geralmente seguido por um elemento ou multiplos <dt> na mesma linha indicam vários termos sendo definidos pelo próximo element <dd>.<dd>;</p>

Exemplo

                        

                            
A description list, with terms and descriptions:

<dl>
    <dt>Coffe</dt>
    <dd>Black hot drink</dd>
    <dt>Milk </dt>
    <dd>White cold drink</dd>
</dl>
