Tag <object>

Definições e uso

A tag <object> define um contêiner para um recurso externo. O recurso externo pode ser uma página da Web, uma imagem, um reprodutor de mídia ou um aplicativo plug-in.
Para incorporar uma imagem, é melhor usar a tag <img>. Para incorporar HTML, é melhor usar a tag <iframe>. Para incorporar vídeo ou áudio, é melhor usar as tags <video> e <audio>.

Exemplo

                        
<!DOCTYPE html>
<html>
    <head>
        <title>Tag object</title>
    </head>
    <body>
    <h1>O elemento object</h1>

    <object data="pic_trulli.jpg" width="300" height="200"></object >

    </body>
</html>

Atributos Padrões

Atributo Valor Descrição
data URL Especifica a URL do recurso a ser usado pelo objeto
form form_id Especifica a qual formulário o objeto pertence
height pixels Especifica a altura do objeto
name name Especifica um nome para o objeto
type media_type Especifica o tipo de mídia dos dados especificados no atributo de dados
typemustmatch true/false Especifica se o atributo type e o conteúdo real do recurso devem corresponder para serem exibidos
usemap #mapname Especifica o nome de um mapa de imagem do lado do cliente a ser usado com o objeto
width pixels Especifica a largura do objeto
