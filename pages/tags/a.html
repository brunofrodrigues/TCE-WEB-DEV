Tag <a>

Definições e uso

A tag <a> é usada para criar um hyperlink em uma página web.

Exemplo

                        
<!DOCTYPE html>
<html>
    <head>
        <title>Exemplo de tag <a></title>
    </head>
    <body>
        <h1>Meu site</h1>
        <p>Bem-vindo ao meu site! Aqui estão alguns links úteis:</p>
        <ul>
            <li><a href="https://www.google.com">Google</a></li>
            <li><a href="https://www.facebook.com">Facebook</a></li>
            <li><a href="https://www.youtube.com">YouTube</a></li>
        </ul>
        <p>Você também pode me encontrar em:</p>
        <ul>
            <li><a href="https://www.linkedin.com">LinkedIn</a></li>
            <li><a href="https://www.twitter.com">Twitter</a></li>>
            <li><a href="https://www.instagram.com">Instagram</a></li>
        </ul>
    </body>
</html>

Atributos Padrões

Atributo Valor Descrição
download filename Especifica que o elemento vai ser instalado quando o hyperlink é clicado pelo usuário
href URL Especifica o URL da página para qual o link leva
hreflang languague_code Especifica a língua da página de destino do hyperlink
media media_query Especifica a midia/dispositivo para qual a página de destino do hyperlink foi optimizada
ping list_of_URLs Especificas uma lista separada por espaço de URLs para as quais, quando o link for seguido, solicitações POST com o corpo "ping" serão enviadas pelo navegador (em segundo plano). Geralmente usado para rastreamento.
referrepolicy no-referrer
no-referrer-when-downgrade
origin
origin-when-cross-origin
same-origin
strict-origin-when-cross-origin
unsafe-url		 Especifica qual informação de referência (referrer) enviar com o link.
rel alternate
author
bookmark
external
help
license
next
nofollow
noreferrer
noopener
prev
search
tag		 Especifica a relação entre o documento atual e o documento vinculado.
target _blank
_parent
_self
_top		 Especifica onde abrir o documento vinculado.
type media_type Especifica o tipo de mídia do documento vinculado.
