Tag <bdo>

Definições e uso

BDO significa Bi-Directional Override. A tag <bdo> é usada para substituir a direção do texto atual.

Exemplo

                        
<!DOCTYPE html>
<html>
    <head>
        <title> Tag bdo</title>
    </head>
    <body>
    <bdo dir="rtl"">
        <O texto agora vai estar 
         na orientação "right-to-left" 
         ou direita para esquerda.>
    </object>

    </body>
</html>

Atributos Padrões

.
Atributo Valor Descrição
dir ltr
rtl		 Sempre requerida, especifica a direção do texto dentro do elemento <bdo>
