Tag <select>

Definições e uso

O <select> elemento é usado para criar uma lista suspensa.
O <select> elemento é usado com mais frequência em um formulário para coletar a entrada do usuário.

O name atributo é necessário para fazer referência aos dados do formulário após o envio do formulário (se você omitir o nameatributo, nenhum dado da lista suspensa será enviado).

O id atributo é necessário para associar a lista suspensa a um rótulo.

As tags <option> dentro do <select>elemento definem as opções disponíveis na lista suspensa.

Dica: Sempre adicione a tag <label> para melhores práticas de acessibilidade!

Exemplo

                        
<!DOCTYPE html>
<html>
<body>
<h1> Selecione o elemento </h1>
<p> O elemento select é usado para criar uma lista suspensa. </p>
<form action="/action_page.php">
<label for="cars">Escolha um carro:</label>
    <select name="carro" id="carro">
        <option value ="volvo"> Volvo </option>
        <option value ="saab"> Saab </option>
        <option value ="opel"> Opel </option>
        <option value ="audi"> Audi </option>
        </select>
        <br> <br>
        <input type="submit" value="Submit">
        </form>
        <p> Clique no botão "Enviar" e os dados do formulário 
        serão enviados para uma página no
        servidor chamado "action_page.php". </p> 
    </body>
</html>

Atributos Padrões

Atributo Valor Descrição
autofocus autofocus Especifica que a lista suspensa deve obter foco automaticamente quando a página for carregada
disabled disabled Especifica que uma lista suspensa deve ser desativada
form form_id Define a qual formulário a lista suspensa pertence
multiple multiple Especifica que várias opções podem ser selecionadas ao mesmo tempo
name name Define um nome para a lista suspensa
required required Especifica que o usuário deve selecionar um valor antes de enviar o formulário
size number Define o número de opções visíveis em uma lista suspensa
