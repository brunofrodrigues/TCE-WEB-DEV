Tag <del>

Definições e uso

A tag <del> é usada para indicar texto deletado ou removido em uma página HTML.

Exemplo

                    
<!DOCTYPE html>
<html>
<head>
    lt;title>Texto Deletado</title>
</head>
<body>
    <h1>Exemplo de Texto Deletado</h1>
                       
    <p>Este é um parágrafo com <del>texto deletado</del>.</p>
</body>
</html>

Atributos Padrões

Atributo Valor Descrição
cite URL Especifica a URL de um documento que explica porque o texto foi deletado/modificado
datetime YYYY-MM-DDThh:mm:ssTZD Especifica a data e o tempo em que o texto foi deletado/modificado
