Tag <link>

Definições e uso

A tag <link>O Elemento HTML <link> especifica as relações entre o documento atual e um recurso externo. Possíveis usos para este elemento incluem a definição de uma estrutura relacional para navegação. Este elemento é mais usado para vincular as folhas de estilo.

Exemplo

                        

                            
Link to an external style sheet:

<head>
    <link rel ="stylesheet" href="styles.css">
</head>

Atributos Padrões

Atributo Valor Descrição
title . Defines a preferred or an alternate stylesheet
type media_type Specifies the media type of the linked document
href URL Specifies the location of the linked document
