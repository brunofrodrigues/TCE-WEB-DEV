Tag <map>

Definições e uso

A tag <map> é usada para definir um mapa de imagem. Um mapa de imagem é uma imagem com áreas clicáveis.

O atributo name obrigatório do elemento <map> está associado ao atributo usemap do <img> e cria um relacionamento entre a imagem e o mapa. O elemento <map> contém vários elementos <area>, que definem as áreas clicáveis ​​no mapa de imagem.

Exemplo

                        
<!DOCTYPE html>
<html>
    <head>
        <title>Tag ul</title>
    </head>
    <body>
    <h1>Os elementos do map e da area</h1>

    <p>Clique no computador, no telefone ou na xícara de café para
     ir para uma nova página e ler mais sobre o assunto:</p>

    <img src="workplace.jpg" alt="Workplace" usemap="#workmap" width="400" 
    height="379">

    <map name="workmap">
     <area shape="rect" coords="34,44,270,350" alt="Computer" href="computer.htm">
     <area shape="rect" coords="290,172,333,250" alt="Phone" href="phone.htm">
     <area shape="circle" coords="337,300,44" alt="Cup of coffee" href="coffee.htm">
    </map>


    </body>
</html>

Atributos Padrões

Atributo Valor Descrição
name mapname Obrigatório.
Especifica o nome do mapa de imagem
