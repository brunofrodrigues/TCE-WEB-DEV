Desenvolvimento WEB

Encontre aqui várias referências de HTML, de fácil acesso e grátis.

HTML5 é uma Markup Language usada para criar e estruturar conteúdo em uma página da web. Ele define a estrutura da página, incluindo textos, imagens, links e outros elementos. O HTML é a base do desenvolvimento web e é interpretado pelos navegadores para exibir o conteúdo de uma página.

CSS3 é uma Style Language que define a aparência visual de uma página WEB. Ele separa o conteúdo da apresentação e é usado com o HTML para criar layouts personalizados e estilos visuais. As regras de estilo do CSS especificam a aparência dos elementos, como cores e posições.

Programas utilizados por aqui

A plataforma Figma é um software de design de interface do usuário (UI) e experiência do usuário (UX) baseado na nuvem. Graças a ele pudemos desenhar o nosso site.

Usado para programar nosso site, o Visual Studio é uma IDE da Microsoft utilizado para criar aplicativos de software que oferece recursos como edição de código, depuração, entre outro. Compatível com várias linguagens de programação, como C++, C# e HTML.
